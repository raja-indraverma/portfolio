Candlestick Pattern Detection

GitHub Link
This application utilizes cutting-edge technology to detect candlestick patterns in financial charts. It integrates YOLOv8 for object detection, Selenium for web automation, and Twilio for notification services. Below is a detailed breakdown of its technical implementation:

1. YOLOv8 Model:

Pretrained Model: The YOLOv8 model, fine-tuned for candlestick patterns, is loaded using the ultralyticsplus library.

  • Confidence threshold: 0.25
  • IoU threshold: 0.45
  • Maximum detections: 1000

2. Web Browser Automation:

Selenium with ChromeDriver automates the loading of financial charts from TradingView. Screenshots of the browser window are captured for analysis.

3. Real-Time Image Processing:

  • Browser screenshots are converted into NumPy arrays for model inference.
  • YOLOv8 processes these images to identify candlestick patterns.

4. Result Visualization:

Detected patterns are overlaid on the original images using YOLO's utilities. The annotated frames are displayed in real-time with OpenCV.

5. Notification System:

  • WhatsApp Integration: Sends alerts to the user with Twilio's API.
  • Sound Notifications: Alerts via system sound (Windows only).

Key Technical Components:

Feature Technology/Library Purpose
Object Detection YOLOv8 Candlestick pattern recognition
Browser Automation Selenium Loading and capturing chart screenshots
Image Processing OpenCV, NumPy Preparing images for inference
Notifications Twilio WhatsApp alerts for detected patterns
Environment Management Python, .env Configuring API keys and environmental variables

Potential Applications: Real-time trading analysis, educational tools for learning technical analysis, and market pattern recognition.
